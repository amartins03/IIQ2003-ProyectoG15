# IIQ2003-ProyectoG15
Repositorio de proyecto IIQ2003 del grupo 15.

En 2024, el salmón representó para Chile el segundo producto de exportación más relevante, 

Este proyecto tiene como objetivo implementar un modelo de transferencia de masa mediante modelación teórica-matemática y programación, permitiendo la optimización de la disponibilidad de oxígeno en una jaula de salmones y reducir el impacto ambiental y mejorar la destinación de recursos en esta industria. Los datos utilizados corresponden a jaulas salmoneras de Puerto Montt, Región de Los Lagos, Chile.
